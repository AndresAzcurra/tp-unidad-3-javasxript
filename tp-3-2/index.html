¿Que es un algoritmo?

Es un conjunto de instrucciones, ordenadas lógicamente de manera que, ejecutándolas en ese orden, se llegue al resultado esperado para resolver/solucionar un problema. Precede al código en lenguaje de programación. Con el pensamiento algorítmico nos planteamos el problema y pensamos la solución. Se pueden expresar con diagramas de flujo. Tiene un inicio, input (datos de entrada), proceso/s, output (resultado) y un fin.

¿Que es un seudocodigo ?

Es el lenguaje de descripción de un algoritmo, expresando en forma detallada cuáles son los pasos que seguirá el programa, pero en lenguaje natural. No existe una forma estándar de escribirlo. Se lo utiliza una vez definido el algoritmo (es decir, una vez que se identificó el problema cómo solucionarlo, o cómo llegar al resultado esperado), y antes de pasarlo a código.
